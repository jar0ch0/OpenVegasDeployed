HOW TO PLAY

OpenVegas quick start for command-line and web play.

Run one command for chat-based gameplay and one command for the browser UI. Use this page as your step-by-step starter guide.

BEGINNER QUICK START

Start here in under 5 minutes

Step 1Go to openvegas.ai, click OPENVEGAS SIGNUP, and confirm your email (check Spam/Junk if needed).
Step 2Install the CLI: npm install -g openvegas@latest
Step 3Open 2 terminal windows: one for openvegas chat (agentic coding/chat) and one for openvegas ui (games + wallet flow).
Step 4Log in with openvegas login when prompted.
Step 5New users receive 100 free $V to get started.
Step 6To buy more credits, go to /ui/balance and click TOP UP NOW.
Step 7Continue in either mode: openvegas chat for coding workflows or openvegas ui for gameplay and balance/history checks.
MAIN COMMANDS

Use these first

openvegas login to log into the account you created.

openvegas chat launches the terminal agentic workflow you're used to.

openvegas ui launches the guided terminal UI mode.

Start with openvegas chat for regular agentic workflow, then use openvegas ui for a guided game flow.

WALKTHROUGH

Suggested first run

Step 1Run openvegas chat to use the agentic terminal workflow.
Step 2Run openvegas ui.
Step 3Top up credits if needed by selecting 'Balance' option.
Step 4Run openvegas ui again and pick a game.
Step 5Play rounds, then return to UI mode to review status.
GAME EXAMPLE

Example flow: Roulette session

1In terminal, run openvegas ui, pick 'Play'.
2Select roulette (or any other game) and place a bet (example: 50 on red).
3Spin resolves; winnings or loss are applied immediately.
4Run openvegas ui to confirm updated balance and recent activity.
