# OpenVegas Beginner Guide

- Go to **https://openvegas.ai**.
- Click **OPENVEGAS SIGNUP**.
- Create your account, then open your email and confirm it (check **Spam/Junk** if you do not see it).

- Install OpenVegas CLI:
  - `npm install -g openvegas@latest`

- Open **2 terminal windows**:
  - Window 1: run `openvegas chat` (agentic coding/chat).
  - Window 2: run `openvegas ui` (games + wallet UI flow).

- Log in when prompted:
  - `openvegas login`
  - If Touch ID is available on your setup, you can use it; otherwise use email/password.

- New users start with **100 free $V** to try the platform.
- In `openvegas ui`, use **Balance** / **History** to see your current funds and activity.

- To buy more credits:
  - Go to **https://app.openvegas.ai/ui/balance**
  - Click **TOP UP NOW**
  - Complete checkout to add more $V credits.

- After top-up, return to `openvegas chat` or `openvegas ui` and continue.
